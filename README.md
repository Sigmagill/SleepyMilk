# SleepyMilk – Smart Sleep Solution

SleepyMilk adalah website landing page yang dikembangkan sebagai media untuk memperkenalkan produk minuman herbal yang berfokus pada peningkatan kualitas tidur. Website ini dirancang dengan pendekatan sederhana namun efektif, sesuai dengan konsep Minimum Viable Product (MVP), yaitu menampilkan nilai utama produk tanpa fitur yang berlebihan. Melalui website ini, pengguna dapat memahami manfaat SleepyMilk, cara kerja produk, serta memberikan feedback berdasarkan pengalaman mereka. Fokus utama dari pengembangan website bukan pada kompleksitas teknologi, melainkan pada bagaimana pengguna dapat dengan cepat memahami value produk dan terdorong untuk mencoba. Website ini juga terintegrasi dengan sistem pencatatan berbasis spreadsheet yang digunakan untuk mengumpulkan data penjualan dan feedback pengguna. Data tersebut kemudian dianalisis untuk mendukung pengembangan produk secara berkelanjutan berdasarkan kebutuhan nyata di lapangan. Dengan pendekatan ini, SleepyMilk tidak hanya menjadi produk, tetapi juga bagian dari proses validasi bisnis berbasis data.

📥 Cara Menggunakan Project (Clone dari GitHub)

Project ini bisa digunakan atau dipelajari dengan cara melakukan clone repository dari GitHub.

🔗 1. Ambil Link Repository
Buka repo kamu di GitHub
Klik tombol Code
Copy link HTTPS, contoh:
https://github.com/username/sleepymilk-website.git
💻 2. Clone ke Laptop

Buka terminal / CMD, lalu jalankan:

git clone https://github.com/username/sleepymilk-website.git
📂 3. Masuk ke Folder Project
cd sleepymilk-website
🌐 4. Jalankan Website

Kalau project kamu HTML biasa:

Klik file index.html
atau
Klik kanan → Open with browser
